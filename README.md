
# 创业游戏

# 运行环境

请先检查ruby版本是否为2.0.0

```ruby
ruby -v
````

如果不是 请参照各系统升级的方法，升级为2.0.0（或以上）


# 运行

运行游戏请执行

```ruby
bundle install # add all ruby libs

ruby game.rb
```
